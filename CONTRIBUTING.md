# Contributing

Thanks for helping grow this list. It aims to stay curated, not exhaustive: entries should be things a researcher or practitioner in explainable NLP would plausibly want to know about.

## What fits

- Peer-reviewed papers and widely cited preprints on explainability and interpretability for NLP and LLMs
- Actively maintained tools and libraries
- Datasets and benchmarks with public access
- Tutorials, courses, workshops, and venues dedicated to the area

## Entry format

Papers:

    - **Title** — Authors, Venue Year. [[paper](https://...)]

Tools and datasets:

    - [Name](https://...) — one-line description

Keep descriptions factual and short. Prefer stable links (arXiv, ACL Anthology, DOI, project pages) over blog mirrors.

## How to contribute

1. Check the entry is not already listed (search the README).
2. Verify the link loads.
3. One entry per pull request is fine; small batches grouped by section are also welcome.
4. Add the entry to the most fitting section, keeping rough chronological order.
5. If unsure, open an issue with the "Suggest an entry" template instead — that works just as well.

New sections: please open an issue first so scope can be discussed. Current plans live in [ROADMAP.md](ROADMAP.md).

The link checker runs on every pull request; a red check usually means a typo in the URL.
